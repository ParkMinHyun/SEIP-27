\subsection{The Evolving Role of the Draft Sequence}
\label{sec:draft-background}

A shutter event produces multiple image buffers. The full post-processing path
collects the required buffers, performs multi-frame composition and image
enhancement, encodes the result, and stores the final image file. The Draft
Sequence was introduced together with this pipeline to produce the first image
shown to the user while full post-processing continues. The Draft image is also
retained as a recovery result if final processing fails. It is therefore both a
user-visible capture result and a reliability mechanism, rather than an
internal preview artifact.

As post-processing became more sophisticated, its greater compute and memory
demand made foreground execution increasingly disruptive to camera usability.
Selected capture modes therefore defer full composition until the camera
application switches to the background. At the same time, advanced processing
produces larger visual changes, widening the gap between untreated Draft and
final images. Because deferral leaves the Draft visible for longer, this
mismatch becomes more apparent. The initial Draft applied no image effects;
single-frame Filter and Watermark were later added to improve consistency.
AI-based post-processing has since widened the gap beyond what these
single-frame stages can reproduce. The mismatch is particularly conspicuous in
portrait capture because the presence or absence of background blur is
immediately recognizable. Figure~\ref{fig:draft-pipeline} illustrates the Draft
and deferred final-processing paths.

Reducing this mismatch requires the portrait Draft Sequence to perform
lightweight multi-frame composition that approximates the final Bokeh effect.
The Draft Sequence, however, must still produce this image quickly because it
serves as both the first user-visible result and the recovery result. Thus,
adding even a lightweight visual effect to the Draft Sequence is not merely a
quality enhancement; it introduces additional computation into a
latency-critical path.
